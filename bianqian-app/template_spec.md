### 配色方案

| 用途 | 颜色值 | 说明 |
|------|--------|------|
| 60% 主色（背景/表面） | {{COLOR_PRIMARY}} | 例：#FAFAF7 暖白 |
| 30% 辅助色（文字/边框） | {{COLOR_SECONDARY}} | 例：#1A1A1A 碳黑 |
| 10% 强调色（按钮/高亮） | {{COLOR_ACCENT}} | 例：#C67A4B 赤陶 |
| 成功色 | {{COLOR_SUCCESS}} | 例：#3A7D1A 森绿 |
| 警告/危险色 | {{COLOR_DANGER}} | 例：#D44444 朱红 |
| 辅助文字色 | {{COLOR_TEXT_2}} | 例：#6B6B6B 中灰 |

> 配色原则：遵循 60-30-10 法则。主色奠定基调，辅助色建立层级，强调色引导操作。文本与背景对比度 ≥ 4.5:1（WCAG 标准）。

### 排版规范

| 用途 | 字体 | 字号 | 字重 |
|------|------|------|------|
| 标题 | {{FONT_HEADING}} | 17-20px | 600 |
| 正文 | {{FONT_BODY}} | 14-16px | 400 |
| 辅助文字 | 同正文 | 12-13px | 400/500 |

> 排版原则：全站不超过 2 种字体。正文行高 ≥ 1.5 倍字号。正文每行不超过 75 个字符。

### 间距与圆角

- **间距基准**：8px 网格（8 / 16 / 24 / 32 / 40）
- **卡片内边距**：{{CARD_PADDING}}（推荐 20-28px）
- **模块间距**：{{MODULE_GAP}}（推荐 16-24px）
- **圆角**：{{BORDER_RADIUS}}（0px 硬朗 / 4-8px 中性 / 12px+ 圆润）

### 必须实现的视觉反馈

| 用户操作 | 视觉反馈要求 |
|---------|------------|
| 创建/新增 | 新元素入场动画（淡入 + 位移），非瞬间出现 |
| 删除 | 淡出或收缩消失，非瞬间移除 |
| 完成/打卡 | 状态变化 + 正向反馈（颜色变化、图标切换） |
| 拖拽放置 | 拖拽中元素半透明，目标区域高亮提示 |
| 悬停可交互元素 | 微抬升或背景变化，明确可点击 |
| 视图切换 | 淡入过渡，不能瞬切 |
| 空状态 | 友好的引导文案，告诉用户下一步做什么 |

### 动效标准

- 所有过渡时长控制在 200-300ms
- 使用 ease 或 cubic-bezier 缓动曲线，禁止线性运动
- 尊重用户的 `prefers-reduced-motion` 系统偏好
- 动效必须有目的（提供反馈 / 引导注意），禁止为了装饰而加动画
